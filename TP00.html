¿Cuàles son los tipos de tìtulos que hay?

Los tipos de titulos que existen son: h1-h2-h3-h4-h5-h6

¿Cuàl es la importancia de nùmero que lleva la etiqueta de titulo

La importancia de nùmero que lleva cada etiqueta de titulo es que dependiendo del numero va a variar el tamaño de la letra. Siendo h1 el formato de letra mas grande y h6 el màs chico

¿Què es lo que debe evitarse para que el uso de los titulos sea accesible? Justifica

Lo que debe evitarse es saltearse los tipos de titulos que hay. Como por ejemplo arrancar con un formato h1 y despuès poner un formato h5

Un atributo en una etiqueta html es un valor que se puede utilizar en cualquier elemento html que configuran o cambian la etiqueta dependiendo de lo que el usuario indique 2 ejemplos: html lang="es" meta charset="UTF-8"

¿Què es el HTML y en què consiste?

El HTML es un lenguaje que define la estructura del contenido de una pàgina web. Consiste en una serie de elementos que le dan la forma a la pàgina web

¿Què es una etiqueta? Funciona si se escribe tanto en minuscula como en mayuscula? Cuàl es la mejor pràctica para escribir etiquetas?

Una etiqueta es una funciòn con la cual podemos cambiar la forma en la que se escribe un texto. Si, funciona si se escribe con minuscula o mayùscula. La mejor pràctica para escribir etiquetas es hacerlo en minùscula para que sea màs entendible

¿A què llamamos elemento HTML? ¿Cuàl es su antomia? Explicar sus partes

Llamamos elemento HTML a la parte que compone la pàgina web, ya sea un texto, una imagen, etc. Su anatomia es:
Etiqueta de apertura : Consiste en el nombre del elemento, envuelto en corchetes. Esta etiqueta marca dónde inicia el elemento
Contenido : Incluye el contenido de la etiqueta (texto)
Etiqueta de cierre : Incluye una barra diagonal delante del nombre del elemento. Esto marca donde termina el elemento
